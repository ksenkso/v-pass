# v-pass

Compile-time directive for passing props through multiple levels of nesting.

> **Attention**
> 
> For everyone who wanted a solution like this - please consider using something else first!
> 
> This is not a production-ready solution and considered a bad practice.

I made this for fun when I saw a component tree in which about 300 props were being passed to lower-levels from a 2500 loc top-level component.
